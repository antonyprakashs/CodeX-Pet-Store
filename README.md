🐾 Tail Spin - Online Pet Store
Tail Spin is a comprehensive full-stack web application designed for pet owners. It serves as a one-stop shop for high-quality pet food, stylish accessories, and professional grooming services. Built by the Code X team, this project demonstrates modern web development practices including responsive design and interactive user interfaces.

🚀 Project Overview
This project was developed as part of a full-stack web development course. It features a clean, user-friendly interface that allows customers to browse products, view grooming details, and book appointments seamlessly across all devices.

Key Features
Product Catalog: Specialized sections for Dogs, Cats, Birds, and Aquarium pets with category-based filtering.

Interactive Grooming Services: A dedicated grooming page with a side-drawer panel that displays service details (Bath & Brush, Nail Clipping, etc.) and pricing.

Appointment Booking: A fully functional modal-based booking system for pet grooming appointments.

Responsive Design: A mobile-first approach featuring a specialized three-dot navigation menu for smaller screens.

User Experience: Interactive elements like a toggleable search bar and user account dropdown menus.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (including Flexbox/Grid), and Tailwind CSS.

Interactivity: Vanilla JavaScript for DOM manipulation, event handling, and form validation.

Fonts & Icons: Google Fonts (Roboto, Poppins) and SVG-based iconography.

📂 Project Structure
index.html: The landing page featuring hero banners, best sellers, and customer reviews.

product.html: The e-commerce hub for pet food and accessories.

grooming.html: The service portal for pet care and bookings.

contact.html: A dedicated page for user inquiries and business location details.

script.js: The engine handling navigation, modals, search, and form submissions.

style.css: Custom styling and responsive media queries for all components.
